JavaScript Tutorial

1. JS Introduction

JavaScript is a programming language that allows you to implement complex features on web pages. It is used for adding interactivity to web pages and creating dynamic content.

2. JS Where To

JavaScript can be included in HTML documents in three ways: Inline, Internal, External.

3. JS Output

Output methods in JS: alert(), console.log(), document.write().

4. JS Statements

A statement in JavaScript performs an action. Example: let x = 5;, myFunction();.

5. JS Syntax

Syntax in JavaScript refers to the rules that define a correctly structured program. Example: semicolons, curly braces {}, and parentheses ().

6. JS Comments

Comments are used to explain code and are not executed. Example: // Single-line comment, /* Multi-line comment */.

7. JS Variables

Variables are containers for storing data values. let, const, and var are used to declare variables.

8. JS Let

let is used to declare variables with block scope, allowing you to update their values.

9. JS Const

const is used to declare variables that are constant and cannot be reassigned.

10. JS Operators

Operators perform operations on variables and values. Examples include: +, -, ==, ===, etc.

11. JS Arithmetic

Arithmetic operators in JS: +, -, *, /, %, ++, --.

12. JS Assignment

Assignment operators assign values to variables. Examples: =, +=, -=, *=, /=.

13. JS Data Types

JavaScript has various data types: String, Number, BigInt, Boolean, undefined, Symbol, null.

14. JS Functions

Functions are blocks of code designed to perform a task. They can be declared using the function keyword or as arrow functions.

15. JS Objects

Objects are collections of key-value pairs. Example: let person = { name: "John", age: 30 };

16. JS Object Properties

Properties are values associated with an object. They can be accessed with dot notation or bracket notation.

17. JS Object Methods

Methods are functions associated with an object. Example: person.greet().

18. JS Object Display

You can display an object using console.log() or by iterating over its properties.

19. JS Object Constructors

Constructors allow you to create multiple objects of the same type. Example: function Person(name, age) { this.name = name; this.age = age; }

20. JS Events

Events are actions that occur in the browser, such as clicks, keypresses, etc., and can be handled using event listeners.

21. JS Strings

Strings are sequences of characters. You can manipulate strings using methods like toUpperCase(), substring(), and replace().

22. JS String Methods

Common string methods: .length, .toLowerCase(), .trim(), .charAt(), etc.

23. JS String Search

JavaScript offers methods like .indexOf(), .search(), .match() to search within strings.

24. JS String Templates

Template literals allow embedding expressions inside strings: Hello, ${name}.

25. JS Numbers

Numbers can be integers or floating-point. JavaScript provides methods like .toFixed(), .toPrecision(), etc.

26. JS BigInt

BigInt is a special type that can represent integers beyond the range of the Number type.

27. JS Number Methods

Common methods include .toString(), .toFixed(), .valueOf().

28. JS Number Properties

Number properties include Number.MAX_VALUE, Number.MIN_VALUE, etc., which provide limits for the number type.

29. JS Arrays

Arrays store multiple values in a single variable. Example: let arr = [1, 2, 3];

30. JS Array Methods

Methods like .push(), .pop(), .shift(), .unshift() modify array elements.

31. JS Array Search

Methods like .indexOf(), .find(), .includes() are used for searching in arrays.

32. JS Array Sort

.sort() is used to sort the elements of an array.

33. JS Array Iteration

Iteration can be done using .forEach(), for loop, map(), etc.

34. JS Array Const

const can be used to declare arrays, but their contents can still be modified.

35. JS Dates

Dates in JavaScript are handled using the Date object.

36. JS Date Formats

Dates can be displayed in various formats using methods like .toDateString(), .toLocaleDateString(), etc.

37. JS Date Get Methods

Methods like .getFullYear(), .getMonth(), .getDate() retrieve parts of a date.

38. JS Date Set Methods

Methods like .setFullYear(), .setMonth(), .setDate() set parts of a date.

39. JS Math

The Math object provides mathematical constants and functions like .pow(), .sqrt(), .random().

40. JS Random

Math.random() generates a random floating-point number between 0 and 1.

41. JS Booleans

Booleans represent true or false values, often used in conditional statements.

42. JS Comparisons

Comparison operators check for equality or inequality, such as ==, ===, !=, >.

43. JS If Else

Conditional statements allow execution of code based on conditions. Example: if (x > 10) { ... } else { ... }

44. JS Switch

switch allows multi-way branching based on the value of a variable.

45. JS Loop For

The for loop repeats a block of code a specified number of times.

46. JS Loop For In

The for...in loop iterates over object properties.

47. JS Loop For Of

The for...of loop iterates over iterable objects like arrays or strings.

48. JS Loop While

The while loop runs as long as the condition is true.

49. JS Break

The break statement exits a loop early.

50. JS Iterables

Iterables are objects that can be iterated over, such as arrays, maps, and sets.

51. JS Sets

Sets store unique values. Example: let set = new Set([1, 2, 3]);

52. JS Set Methods

Methods like .add(), .delete(), .has() are used with sets.

53. JS Maps

Maps store key-value pairs and allow any type of key. Example: let map = new Map();

54. JS Map Methods

Methods like .set(), .get(), .has() are used with maps.

55. JS Typeof

The typeof operator is used to determine the type of a variable.

56. JS Type Conversion

JavaScript allows converting between data types using methods like String(), Number(), and Boolean().

57. JS Destructuring

Destructuring allows unpacking values from arrays or properties from objects into distinct variables.

58. JS Bitwise

Bitwise operators work on binary representations of numbers, such as &, |, ^, ~, <<, >>.

59. JS RegExp

Regular expressions are used for pattern matching in strings.

60. JS Precedence

Operator precedence determines the order in which operations are performed.

61. JS Errors

JavaScript has different types of errors like SyntaxError, ReferenceError, and TypeError.

62. JS Scope

Scope refers to the visibility and lifetime of variables. It can be global or local (block-level).

63. JS Hoisting

Hoisting is JavaScript's behavior of moving variable and function declarations to the top of their scope.

64. JS Strict Mode
